# curso-fullstack

App de controle de treino.

## Requisitos

- Node.js 20+ (recomendado)
- npm 10+

## Instalação

```bash
npm install
```

## Dependências de desenvolvimento

- TypeScript
- ESLint
- Prettier
- Tipagens do Node.js

## Scripts disponíveis

Atualmente, o projeto possui apenas o script padrão de teste do npm:

```bash
npm test
```

## Próximos passos

- Criar `tsconfig.json`
- Adicionar scripts de `lint`, `format` e `build` no `package.json`
- Estruturar a pasta `src/`
